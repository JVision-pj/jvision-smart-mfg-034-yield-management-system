# 良率管理管理艙

整合製程、測試與缺陷數據，分析與提升產品良率之系統，常見於半導體與PCB產業。

- 原始系統：良率管理系統（Yield Management System）
- 產業分類：智慧製造｜製造管理
- 使用部門：良率工程、製程工程、品保
- 工作流程：測試/缺陷數據收集 → 良率統計分析 → 根因歸因 → 改善對策 → 效果驗證追蹤
- AI 應用：AI辨識缺陷圖形模式（Pattern Recognition）、機器學習預測良率風險批次
- 主要 KPI：良率、缺陷密度、良率損失金額
- 代表產品：PDF Solutions Exensio、Camstar Yield Management、yieldWerx
